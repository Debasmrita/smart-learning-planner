# 📚 StudyBuddy – Smart Learning Planner

StudyBuddy is a **full-stack productivity web application** designed to help students organize their study tasks, track progress, and maintain consistent learning habits.

The application allows students to create study topics, manage priorities, monitor their study journey, track progress statistics, and manage their profile securely.

## Live Demo:
https://smart-learning-planner-rgrj-git-main-debasmritas-projects.vercel.app

---

# 🚀 Features

## 🔐 Authentication System

* Student **registration and login**
* **JWT-based authentication**
* Secure password storage using **bcrypt**
* Protected routes for authenticated users

---

## 📊 Smart Study Dashboard

The dashboard provides a quick overview of learning progress.

Features include:

* Total study topics count
* Completed topics count
* Progress percentage
* Study streak tracking
* Suggested topic insights

Students can easily monitor their productivity and progress.

---

## ✨ Quick Task Creator

Students can quickly add new study tasks.

Options include:

* Topic name
* Priority level (High / Medium / Low)

This allows students to manage their study workflow efficiently.

---

## 🧠 Study Journey Tracker

The **Study Journey section** shows all study topics with actions to manage them.

Students can:

* Mark topics as **completed**
* Undo completed topics
* Delete topics
* View topic priority

This creates a structured **learning workflow manager**.

---

## 🏆 Achievement Dashboard

The dashboard also provides study insights like:

* Today's tasks
* Weekly tasks
* Overall progress

This helps students stay motivated and track their consistency.

---

## 🔥 Study Streak

StudyBuddy keeps track of **daily study streaks**, encouraging students to maintain continuous learning habits.

---

## 👤 Profile Management

Students can manage their profile easily.

Features include:

* View account information
* Update name
* Change password securely
* Logout functionality
